### Quick Setup Guide

Simply 'Drag-n-Drop' the SCDefault prefab into a scene. Create a UI EventSystem game object within your scene.  

An event system is required because the default implementation of the SimpleCommands uses the Unity UI to create and interact with the console.  

> Note: The EventSystem must exist on any scene where the console will be used.
